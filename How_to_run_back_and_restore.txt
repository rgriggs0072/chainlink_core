To backup the application into the chainlink_core_backup folder

Step 1) Close the VS solution
Step 2) open windows powershell
Step 3) navigate to chainlink_core folder (cd "E:\Development\chainlink_core")
Step 5) Run the following code in powershell
	.\backup_chainlink.ps1 -DestRoot "E:\Development\chainlink_core_backups" -Zip

Step 6) Verify you have a folder like (chainlink_core_20251106_205726.zip) and one that is not Zipped


To restore from back if need do the following

restore_chainlink.ps1

Restores the latest backup (folder or .zip) from E:\Development\chainlink_core_backups into your working folder E:\Development\chainlink_core. It also snapshots your current project to a pre_restore_YYYYMMDD_HHMMSS folder so you can undo.

How to use

From E:\Development\chainlink_core:

Restore the newest backup:

.\restore_chainlink.ps1


Restore a specific backup (folder or zip):

.\restore_chainlink.ps1 -BackupPath "E:\Development\chainlink_core_backups\chainlink_core_20251106_204858"
# or
.\restore_chainlink.ps1 -BackupPath "E:\Development\chainlink_core_backups\chainlink_core_20251106_204858.zip"
